Welcome to Azure Blob Explorer's documentation!
===============================================

A Python 3 client to download, upload or delete files or folders from Azure Blob storage. This library also
has a CLI that can be used in Docker or on your local system to pre-download the files.

Instillation
------------

From PyPi:

.. code-block::

   pip install azblobexplorer

From source

.. code-block::

    python setup.py install
